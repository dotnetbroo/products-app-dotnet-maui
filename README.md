# 🛍 Products App (MAUI + ASP.NET Core API)

A simple full-stack application built using **.NET MAUI** and **ASP.NET Core Web API**.  
This project demonstrates how a mobile/desktop app can communicate with a backend service.

---

## 🚀 Features
- 📦 View product list  
- ➕ Add new product  
- ❌ Delete product  
- 🔄 Refresh data from API  

---

## 🛠 Tech Stack
- ASP.NET Core Web API (.NET 9)
- .NET MAUI
- MVVM Pattern
- HttpClient
- Dependency Injection
